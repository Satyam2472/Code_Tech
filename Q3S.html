Check if strings are rotations of each other or not

Discription

Given two strings s1 and s2. The task is to check if s2 is a rotated version of the string s1. The characters in the strings are in lowercase.

Category: Easy.

Input:

First line contains an integer denoting the test cases 'T'. T testcases follow. Each testcase contains two lines having two strings of alphabets s1, s2.

Output:

For each testcase, print the yes or no, whether the string s2 is a rotated version of s1 or not.

Constraints:

Expected time complexity: O(N).
Expected space complexity: O(N).
1 <= T <= 100
1 <= |s1|, |s2| <=105

Test Cases

Input:

2
geeksforgeeks
forgeeksgeeks
mightandmagic
andmagicmigth

Output:

yes
no
