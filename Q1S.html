Reverse a String

Discription

You are given a string s. You need to reverse the string.

Category: Easy.

Input:

First line contains an integer denoting the test cases 'T'. T testcases follow. Each testcase contains a string of alphabets.

Output:

For each testcase, print the given string in reverse order.

Constraints:

Expected time complexity: O(|S|).
Expected space complexity: O(1).
1 <= T <= 100
1 <= |S| <=103

2
