Detect Loop in linked list

Discription

Given a linked list of N nodes. The task is to check if the linked list has a loop. Linked list can contain self loop.

Category: Medium.

Input:

First line contains an integer denoting the test cases 'T'. T testcases follow. Each testcase contain two line, first line the number of elements present in the linked list, and second line contains the value itself.

Output:

For each testcase, print true if there is cycle otherwise false.

Constraints:

Expected time complexity: O(N).
Expected space complexity: O(1).
1 <= T <= 100
1 <= N <=104
1 <= Data_of_linked_list <= 103

Test Cases

Input:

2
3
1 3 4
position at which tail is connected is 2
4
1 8 3 4
position at which tail is connected is -1

Output:

true
false
