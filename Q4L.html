Add 1 to a number represented as linked list

Discription

A number N is represented in Linked List such that each digit corresponds to a node in linked list. You need to add 1 to it.

Category: Easy.

Input:

First line contains an integer denoting the test cases 'T'. T testcases follow. Each testcase contain the elements of the linked list.

Output:

For each testcase, print value after adding 1 to it.

Constraints:

Expected time complexity: O(N).
Expected space complexity: O(1).
1 <= T <= 100
1 <= N <=230

Test Cases

Input:

2
4 5 6
8 9 6 3

Output:

457
8964
