Palindrome String

Discription

Given a string S, check if it is palindrome or not.

Category: Easy.

Input:

First line contains an integer denoting the test cases 'T'. T testcases follow. Each testcase contains a string of alphabets.

Output:

For each testcase, print the yes or no, whether the string is plaindrome or not.

Constraints:

Expected time complexity: O(|S|).
Expected space complexity: O(1).
1 <= T <= 100
1 <= |S| <=104

Test Cases

Input:

2
Code_Tech
abba

Output:

no
yes
