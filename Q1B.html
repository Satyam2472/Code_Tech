Minimum element in BST

Discription

Given a Binary Search Tree. The task is to find the minimum element in this given BST.

Category: Easy.

Input:

First line contains an integer denoting the test cases 'T'. T testcases follow. Each testcase your are given the pre-order traversal.

Output:

For each testcase, print the minimum element presnt in the tree.

Constraints:

Expected time complexity: O(Height of the BST).
Expected space complexity: O(Height of the BST).
1 <= T <= 100
1 <= No_of_elements <=104

Test Cases

Input:

2
5 4 3 1 6 7
9 10 11

Output:

1
9
