Reverse Level Order Traversal

Discription

Given a binary tree of size N, find its reverse level order traversal. ie- the traversal must begin from the last level.

Category: Easy.

Input:

First line contains an integer denoting the test cases 'T'. T testcases follow. Each testcase your are given the pre-order traversal.

Output:

For each testcase, print the tree using reverse level order traversal.

Constraints:

Expected time complexity: O(N).
Expected space complexity: O(N).
1 <= T <= 100
1 <= No_of_nodes <=105
1 <= Dates_of_nodes <=105

Test Cases

Input:

2
1 3 2
10 20 40 60 30

Output:

3 2 1
40 60 20 30 10
