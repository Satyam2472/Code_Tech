Reverse a linked list

Discription

Given a linked list of N nodes. The task is to reverse this list.

Category: Easy.

Input:

First line contains an integer denoting the test cases 'T'. T testcases follow. Each testcase contains the elements present in the linked list.

Output:

For each testcase, print the given linked-list in reverse order.

Constraints:

Expected time complexity: O(N).
Expected space complexity: O(1).
1 <= T <= 100
1 <= N <=104

Test Cases

Input:

2
1 2 3 4 5 6
2 7 8 9 10

Output:

6 5 4 3 2 1
10 9 8 7 2
