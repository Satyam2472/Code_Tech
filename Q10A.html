Overlapping Intervals

Discription

Given a collection of Intervals, the task is to merge all of the overlapping Intervals.

Category: Medium.

Input:

First line contains an integer denoting the test cases 'T'. T testcases follow. Each testcase contains the set of intervals.

Output:

Returns sorted list of intervals after merging.

Constraints:

Expected time complexity: O(N*log(N)).
Expected space complexity: O(log(N)).
1 <= T <= 100
1 <= N <=102
1 <= x,y <=103

Test Cases

Input:

2
{{1,3},{2,4},{6,8},{9,10}}
{{6,8},{1,9},{2,4},{4,7}}

Output:

{{1, 4}, {6, 8}, {9, 10}}
{{1, 9}}
