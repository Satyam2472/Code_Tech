Look and Say Pattern

Discription

Given an integer n. Return the nth row of the following look-and-say pattern.
1
11
21
1211
111221
Look-and-say

To generate a member of the sequence from the previous member, read off the digits of the previous member, counting the number of digits in groups of the same digit. For example:


  • 1 is read off as "one 1" or 11.

  • 11 is read off as "two 1s" or 21.

  • 21 is read off as "one 2, then one 1" or 1211.

  • 1211 is read off as "one 1, one 2, then two 1s" or 111221.

  • 111221 is read off as "three 1s, two 2s, then one 1" or 312211.

Category: Easy.

Input:

First line contains an integer denoting the test cases 'T'. T testcases follow. Each testcase contains a number N representing the corresponding line of look-and-say table.

Output:

For each testcase, print the Nth line of look-and-say table.

Constraints:

Expected time complexity: O(2N).
Expected space complexity: O(2N-1).
1 <= T <= 100
1 <= N <=30

Test Cases

Input:

2
5
3

Output:
