Union of two arrays

Discription

Given two arrays A[] and B[] of size n and m respectively. The task is to find union between these two arrays. Union of the two arrays can be defined as the set containing distinct elements from both the arrays. If there are repetitions, then only one occurrence of element should be printed in the union.

Category: Easy.

Input:

First line contains an integer denoting the test cases 'T'. T testcases follow. Each testcase contains two lines of input. First line contains n and m the size of the array A and B. The second line contains the elements of the array.

Output:

For each testcase, print the length of the array fromed after union of two arrays.

Constraints:

Expected time complexity: O((n+m)log(n+m)).
Expected space complexity: O(n+m).
1 <= T <= 100
1 <= n, m <=105
10-5 <= Ai, Bi <= 105

Test Cases

Input:

2
5 3
1 2 3 4 5
1 2 3
6 2
85 25 1 32 54 6
85 2

Output:

5
7
