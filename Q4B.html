Binary Tree to BST

Discription

Given a Binary Tree, convert it to Binary Search Tree in such a way that keeps the original structure of Binary Tree intact.

Category: Easy.

Input:

The first line of input contains an integer T denoting the number of test cases. Then T test cases follow. Each test case contains has a pre-order traversal of the tree.

Output:

Print the pre-order traversal of the BST formed.

Constraints:

Expected time complexity: O(log(N)).
Expected space complexity: O(N).
1 <= T <= 100
1 <= No_of_nodes <=103

Test Cases

Input:

1 2 3
1 2 4 3

Output:

2 1 3
3 2 1 4
