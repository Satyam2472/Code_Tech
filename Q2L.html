Reverse a Linked List in groups of given size.

Discription

Given a linked list of size N. The task is to reverse every k nodes (where k is an input to the function) in the linked list. If the number of nodes is not a multiple of k then left-out nodes, in the end, should be considered as a group and must be reversed (See Testcase 2 for clarification).

Category: Medium.

Input:

First line contains an integer denoting the test cases 'T'. T testcases follow. Each testcase contain two line, first line the elements present in the linked list, and second line the value of k.

Output:

For each testcase, print the given linked-list in as requirements given in the discription.

Constraints:

Expected time complexity: O(N).
Expected space complexity: O(1).
1 <= T <= 100
1 <= N <=104
1 <= k <= N

Test Cases

Input:

2
1 2 3 4 5 6 7 8
4
1 2 3 4 5
3

Output:

4 2 2 1 8 7 6 5
3 2 1 5 4
